# 可调节网格切分设计

## 目标

让画布编辑器的网格切分工具支持在图像上直接拖动水平或垂直分割线，并在属性栏中即时预览最终会生成的切片。

## 范围

- 保留现有行数、列数和间距输入。
- 切换到网格切分工具时，按当前行列数生成等分的内部水平与垂直分割线。
- 用户可拖动任意内部线；拖动影响该线两侧的单元格，不改变行列数量。
- 切分边界以源画布像素坐标保存。每条线被约束在相邻边界之间，保证所有切片至少有 1 个源像素宽或高。
- 间距沿每个最终单元格的四边内缩，沿用当前语义。
- 属性栏增加预览区：按行列展示所有切片的缩略图，并标注切片序号和实际像素尺寸。
- 点击“切分为 N 帧”使用自定义边界生成帧，保持行优先顺序及既有替换/加入素材库行为。

## 不在范围内

- 自由添加、删除或交换分割线。
- 单独为每条线提供数值输入。
- 在视频网格拆分器中复用本次交互。

## 架构

`gridExtractor` 扩展为接受可选的水平、垂直边界数组，并提供规范化与基于边界创建矩形的纯函数。边界数组含首尾边缘（0 和图像宽/高），内部值必须递增。未提供自定义边界时，工具继续按现有等分算法初始化，确保既有调用不变。

`CanvasEditorTool` 维护两组内部边界状态。改变行、列、图像尺寸时以等分值重新初始化；拖动时仅更新相应轴的内部值。覆盖层改为可交互的线和拖动手柄；单元格遮罩保持非交互。预览切片从现有画布导出显示，不写入项目状态，正式点击切分时再生成最终帧。

## 交互与状态规则

1. 网格工具激活时显示蓝色可拖拽分割线；其余工具保持原有画布事件行为。
2. 指针坐标换算为源像素，并依据缩放比例更新对应边界。
3. 内部边界限制在前后边界之间的至少 1px 范围；指针释放、离开或取消后结束拖动。
4. 行列输入变化将重置该轴为新的等分边界；间距变化不重置边界。
5. 每个预览卡片与最终切分均从同一组计算出的矩形读取，因此尺寸和顺序严格一致。
6. 间距过大时仍保持每个矩形最小 1px，与当前工具的容错行为一致。

## 测试

- `gridExtractor` 单测覆盖：等分默认值、自定义边界、递增/越界边界规范化、间距内缩，以及最终切片尺寸。
- 为边界拖动坐标转换和相邻边界限制抽出纯函数并单测。
- 手工验证：缩放、平移后拖动线；编辑已有帧和单一资产的切分；中文与英文文案；预览与生成帧的数量、尺寸和顺序一致。

## 验收标准

- 用户能拖动任意内部水平或垂直线，视觉网格和预览立即刷新。
- 生成的帧严格遵循拖动后的切分位置及间距。
- 未拖动时，现有等分结果保持不变。
- 不出现零尺寸、重叠或越出源画布的切片。
