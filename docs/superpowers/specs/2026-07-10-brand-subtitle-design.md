# Brand Subtitle Design

## Goal

Align the product's visible subtitle and README positioning with Frame Forge's role as a dynamic sticker creation tool.

## Scope

- Keep the product name and technical package name as `Frame Forge` and `frame-forge`.
- Change the Chinese subtitle to `动态表情制作工具`.
- Change the English subtitle to `Dynamic Sticker Studio`.
- Update the README introduction to lead with dynamic sticker creation rather than generic frame extraction.

## Non-goals

- No component, route, feature, or package-name changes.
- No historical design-document rewrites.

## Verification

Check the localization strings and README content, then run the production build to validate the affected UI source.
